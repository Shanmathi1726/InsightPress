# InsightPress

InsightPress is a reader engagement intelligence platform designed for modern digital publishing and media analytics.

The platform helps publishers understand audience interaction through engagement analytics, publishing trends, reader behavior insights, and interactive dashboards.

---

## Features

- Interactive Analytics Dashboard
- Reader Engagement Intelligence
- Publishing Trend Analysis
- KPI Monitoring
- Category Performance Tracking
- Author Performance Insights
- Executive Recommendation System
- Dark Corporate UI

---

## Modules

### Home
Modern branded landing page for the platform.

### About
Overview of platform objectives and publishing intelligence capabilities.

### Media Source
CSV upload portal for structured analytics datasets.

### Analytics
Interactive dashboards, KPI tracking, trend charts, and article performance analysis.

### Insights
AI-style executive insights and publishing recommendations.

---

## Technologies Used

- Python
- Streamlit
- Pandas
- Plotly
